# SlyOS Native Apple App

This is the editable native Apple wrapper for the SlyOS shell.

It intentionally uses the same built Web UI as `platforms/desktop-shell`, then runs it inside WebKit:

- macOS: native `.app` built locally by `npm run macos:app`
- iPhone/iPad: Xcode project generated by `npm run apple:xcode`

## Edit The UI

Edit the shared shell in:

```text
platforms/desktop-shell/src
```

Then rebuild/sync:

```bash
npm run apple:sync-web
```

## Run On This Mac

```bash
npm run macos:app
open platforms/macos/build/SlyOS.app
```

The Mac wrapper opens as a borderless full-frame SlyOS surface and places the SlyOS bottom nav over the bottom of the Mac display.

## Open In Xcode

```bash
npm run apple:xcode
open platforms/apple/SlyOSNative/SlyOSNative.xcodeproj
```

Select `SlyOS-iOS`, choose the connected iPhone, set your signing team, and press Run.

Full Xcode is required for cabled iPhone install. Command Line Tools alone cannot deploy to an iPhone.

## Icons

The SlyOS app icon is generated from the same orange brain-chip mark used by the PWA:

```bash
npm run apple:icons
```

This refreshes `Assets.xcassets/AppIcon.appiconset`, the macOS `.icns` source iconset, and PNG PWA icons.

## Supabase

Public native resources are built without project-specific Supabase defaults. Use the app's Setup screen to paste the project URL and publishable key, or run a local dev build with your own ignored `.env`.
